# uwufy

## Webpage
https://miha53cevic.github.io/uwufy/

## Inspiration
https://github.com/Dko1905/uwufy

## Compile
- Required g++ for Makefile
```bash
    make
    cd build
    ./uwufy
```

## Usage
```bash
    ./uwufy Hello
    hewwo
```
### Filemode support
```bash
    ./uwufy --file text.txt
    hewwo
```

### Sending output to a textfile
```bash
    ./uwufy Hello > text.txt
```
